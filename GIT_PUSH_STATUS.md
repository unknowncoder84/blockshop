2. **Go to Settings**
   - Click your profile picture (top right)
   - Click "Settings"

3. **Developer Settings**
   - Scroll down to "Developer settings" (bottom left)
   - Click "Personal access tokens"
   - Click "Tokens (classic)"

4. **Generate Token**

## 🆘 Troubleshooting

### Problem: "Authentication failed"
**Solution:** Make sure you're using the token (not password)

### Problem: "Repository not found"
**Solution:** Create repository first at https://github.com/new

### Problem: "Permission denied"
**Solution:** Token needs `repo` scope - generate new token

### Problem: Token doesn't work
**Solution:** Try GitHub Desktop instead:
1. Download: https://desktop.github.com/
2. Login with GitHub
3. Add local repository
4. Publish to GitHub
